## About Allwinner FEL 

FEL is a low-level subroutine contained in the [BootROM](https://linux-sunxi.org/BROM) on Allwinner devices. It is used for initial programming and recovery of devices using USB.

## Depends
- Qt6
- libusb
- xfel
